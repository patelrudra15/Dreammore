PMEGP
PMEGP

Prime Minister's Employment Generation Programme

Apply Now →Details

About

Collateral-free loan for setting up new micro-enterprises in manufacturing and service sectors. Up to ₹50 Lakh with 15-35% capital subsidy.

Eligibility

  • ✓ Age 18 years and above
  • ✓ Class 8 pass for projects above ₹10L
  • ✓ New enterprise only
  • ✓ No income ceiling

Documents

  • ✓ Aadhar & PAN
  • ✓ Caste certificate (if applicable)
  • ✓ Detailed Project Report (DPR)
  • ✓ Bank account details
CGTMSE
CGTMSE

Credit Guarantee Fund Trust for MSEs

Apply Now →Details

About

Government-backed credit guarantee for micro and small enterprises to access institutional credit without collateral up to ₹5 Crore (75-85% cover).

Eligibility

  • ✓ Micro & Small Enterprises
  • ✓ Manufacturing or service sector
  • ✓ Valid MSME/Udyam Registration
  • ✓ Existing or new business

Documents

  • ✓ Udyam Certificate
  • ✓ CA-certified Balance Sheet
  • ✓ IT Returns (3 years)
  • ✓ Project Report
MUDRA
Mudra Loan

Micro Units Development & Refinance Agency

Apply Now →Details

About

Loans under Shishu (₹50K), Kishor (₹5L), and Tarun (₹10L) categories for micro businesses. Low interest, collateral-free.

Eligibility

  • ✓ Non-corporate micro/small enterprises
  • ✓ Business expansion or new setup
  • ✓ No default with any bank
  • ✓ Clean CIBIL record

Documents

  • ✓ Business plan/Quotation
  • ✓ Identity & Address proof
  • ✓ Bank statements (6 months)
  • ✓ Proof of business activity

Stand-Up India

Stand-Up India Scheme

Apply Now →Details

About

Bank loans between ₹10 lakh and ₹1 crore to SC/ST and women entrepreneurs for setting up greenfield enterprises.

Eligibility

  • ✓ SC/ST entrepreneur or woman entrepreneur
  • ✓ First generation entrepreneur
  • ✓ Greenfield enterprise only
  • ✓ Age 18 and above

Documents

  • ✓ SC/ST certificate / Gender proof
  • ✓ Detailed Project Report (DPR)
  • ✓ Identity & Address proof
  • ✓ Land/premises proof
STARTUP
Startup India

Startup India Seed Fund Scheme

Apply Now →Details

About

Seed funding for startups to validate proof of concept and prototype development. Grants up to ₹20L and investments up to ₹50L.

Eligibility

  • ✓ DPIIT recognized startup
  • ✓ Incorporated within last 10 years
  • ✓ Annual turnover below ₹100 Cr
  • ✓ Working on innovation/IP

Documents

  • ✓ DPIIT Recognition Certificate
  • ✓ Incorporation Certificate
  • ✓ Pitch Deck / Business Plan
  • ✓ Financial Statements
PMFME
PMFME

PM Formalisation of Micro Food Processing Enterprises

Apply Now →Details

About

Support for micro food processing enterprises with credit-linked capital subsidy (35%) up to ₹10 Lakh.

Eligibility

  • ✓ Existing micro food processing units
  • ✓ Proprietorship, partnership, SHG, FPO
  • ✓ Annual turnover below ₹3 Cr
  • ✓ Valid FSSAI license

Documents

  • ✓ FSSAI License
  • ✓ Udyam Registration
  • ✓ Project Report with cost estimate
  • ✓ Bank KYC & Land documents
